# EaSynthesizer
A JavaScript class with a very simple synthesizer using Web Audio API.
Work in progress . . .

## How to use
```javascript
import EaSynthesizer from "@/util/audio/EaSynthesizer.js";
const mySynth = new EaSynthesizer("triangle");
mySynth.playTone("C3");
```
